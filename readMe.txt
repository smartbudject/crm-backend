Перед запуском

 - в application.yml файле ввести путь к папке в которой будут храниться картинки.
   строчка upload.path: ${UPLOAD_PATH:<абсолютный путь до папки>}
   При запуске приложения, будет попытка папку создать если ее не существует, если не полчится - приложение упадет
   ПАПКА ОБЯЗАТЕЛЬНО ДОЛЖНА НАХОДИТСЯ ВНЕ ПРОЕКТА.
   
 - порт на котором будет доступна БД стоит 5555. 
   Если надо поменять то заменить 5555 на нужный порт, в местах:
		- в docker-compose.yaml, строки 
		                       ports:
									- "5555:5432"
	    - в application.yml, строка  url: jdbc:postgresql://localhost:5555/catering
- запуск докера
- запуск 
- localhost:8080/
